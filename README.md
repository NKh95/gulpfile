# gulpfile by N.Kh.

>###### Version: 1.0.0


## Requirements
>###### Node.js 8
>###### Gulp 4



## Installation
BASH:

	chmod +x install.sh
	
	./install.sh


## Usage
Browser-sync and build SASS/SCSS to CSS default:

	gulp				//result in folder: app

SASS/SCSS to CSS:

	gulp style 			//result in folder: app/css

Build:

	gulp build 			//result in folder: dist

View build result:

	gulp view
