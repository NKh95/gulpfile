# gulpfile by N.Kh.

>###### Version: 1.0.4


## Requirements
>###### Node.js 8.x.x
>###### NPM 3.5.x
>###### Gulp 4.x.x
>###### BASH 4.x



## Installation
BASH:

	chmod +x install.sh
	
	./install.sh


## Usage
Browser-sync and build SASS/SCSS to CSS default:

	gulp

SASS/SCSS to CSS:

	gulp style

Build:

	gulp build

View build result:

	gulp view
