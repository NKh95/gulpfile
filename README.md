## gulpfile by N.Kh. (CATASTROPHE)

#### Requirements
> **Node.js** >= 8.x.x
>
> **NPM** >= 3.5.x
>
> **BASH** >= 4.x
>
> **Gulp** 4.x.x

#### Installation

**BASH**
>
>	chmod +x install.sh
>	
>	./install.sh

#### Usage
>`gulp` - Browser-sync and build SASS/SCSS to CSS default.
>
>`gulp style` - SASS/SCSS to CSS.
>		
>`gulp build` - Build.
>	
>`gulp test` - View build result.
>
>`gulp clean` - Clean dist directory.
